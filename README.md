# Route-Weather-Planner
A Python desktop application that calculates routes between two locations using the Mapy.cz API and provides weather forecasts at the start, midpoint, and destination using the OpenWeatherMap API. Added database functionality to store user routes. Added UI elements to display the route and weather information.


## API
You have to create .env with your own API keys for both Mapy.cz and OpenWeatherMap. The .env file should look like .env .example
